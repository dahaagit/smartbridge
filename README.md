# 📖 Overview

The **Flood Prediction System** is a Machine Learning-based web application that predicts the likelihood of flooding using environmental and weather-related parameters.

The application provides users with a simple and interactive interface where they can enter required input values and instantly receive a flood prediction.

This project demonstrates the practical application of Machine Learning in disaster management and early warning systems.

# ⚙️ Installation

## 1️⃣ Clone the repository

```bash
git clone https://github.com/USERNAME/FloodPrediction.git
```

## 2️⃣ Navigate to the project

```bash
cd FloodPrediction
```

## 3️⃣ Create a virtual environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 4️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

## 5️⃣ Run the application

```bash
python app.py
```

Visit

```
http://127.0.0.1:5000
```

# 🧠 How It Works

2. Data is preprocessed.
3. The trained Machine Learning model analyzes the inputs.
4. The prediction is generated.
5. The result is displayed instantly on the web page.
